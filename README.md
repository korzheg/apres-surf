# APRES SURF ПЛЁС — landing v5

Что изменено в этой версии:
- переработан hero-блок: возвращён круглый логотип, добавлена новая вертикальная фотография с лодкой, солнцем и башней, выстроена аккуратная композиция для desktop и mobile;
- подвал собран в компактную строку по примеру: логотип, разделитель, подпись `wake weekend`, Telegram и Instagram с подписями;
- поправлен мобильный скролл программы: первая карточка больше не прижимается к краю экрана;
- исправлен конфликт декоративных волн с заголовком «Программа» на узких экранах;
- карточки дней программы стали кликабельными: открывается popup с подробностями, переключением по дням, поддержкой стрелок клавиатуры и свайпа на мобильных;
- добавлен отдельный блок «Калькулятор поездки» с выбором типа номера и количества человек, расчётом цены за человека и итоговой стоимости;
- добавлены новые ассеты: `event-logo-round.webp` и `hero-ples-photo.webp`.

Загрузите папку целиком на Vercel или откройте `index.html` локально.
