# APRES SURF ПЛЁС landing

Откройте `index.html` в браузере.  
Проект автономный: все используемые изображения лежат в папке `assets`.

Исправлено в этой версии:
- hero-текст APRES SURF ПЛЁС используется как графический ассет, а не как системный шрифт;
- main-изображения используются только для верхнего hero-блока;
- в программе используются 4 отдельные фотографии;
- убраны лишние пустоты перед подвалом;
- поправлены адаптив и масштабирование карточек/фотографий.
